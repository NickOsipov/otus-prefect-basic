# OTUS. Prefect. Basic example

This repository contains a simple example of a Prefect flow that demonstrates how to use tasks, flows and deployment.

## Installation

To run this example, you need to have Python installed on your machine. You can install the required dependencies using uv:

```bash
uv sync
```

## Running the Flow

To run the flow, you can use the following command:

```bash
python3 flows/simple.py
```

## Deployment

To deploy the flow, you can use the following command:

```bash
python3 deploments/simple_deploy.py
```
